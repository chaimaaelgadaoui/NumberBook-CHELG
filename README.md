# 📱 NumberBook

**NumberBook** est une application Android permettant de gérer, synchroniser et rechercher des contacts de manière centralisée via un serveur distant.

---

## Contexte du projet

Ce projet a été réalisé dans le cadre du module :

👉 **Sécurité des Applications Multimédia et Embarquées**

L’objectif de ce lab est de mettre en œuvre :
- La gestion sécurisée des données utilisateurs
- La communication client-serveur
- La centralisation et la synchronisation des contacts

---

## Fonctionnalités

- 📥 **Chargement Local** : Récupération sécurisée des contacts du téléphone (avec gestion des doublons)
- 🔄 **Synchronisation** : Envoi des contacts vers une base de données MySQL via une API PHP
- 🔍 **Recherche Intelligente** : Recherche par nom ou numéro avec filtrage côté client
- 🧠 **Architecture** : Utilisation du pattern Singleton avec Retrofit

---

## Stack Technique

- **Langage** : Java  
- **Réseau** : Retrofit 2 & Gson  
- **Interface** : ConstraintLayout, RecyclerView, Material Design  
- **Backend** : PHP / MySQL
